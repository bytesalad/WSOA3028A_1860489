Mbeki's Swing!

Website Style Guide

Last Changed: 04/06/2020

Colour Palette

I chose this colour palette for my website's design because the combination of colours may give the website a noir/retro feel. This is important because part of the website is the life and ideas of a fictional Jazz musician called Johnathan Mbeki. The story covered by the blog posts on this website takes place during the 1950s. Thus the colour decisions should resonate with the feel of the blog posts.

In light of feedback I received about the old colour palette, I have changed it to be more sepia to resemble colour tones of old books or old images. I feel this resonates with the feel of the website more.

Annotated Wireframes - Red marking = new annotation

All blogs will now be accessible from their respective blog main pages only. I made this change because it would not make sense to have a random new blog out of context on the main page especially when the content on this site is so diverse.

I have added a separate navigation link for all of Johnathan Mbeki's blogs so that the difference between my theory and character blogs is defined through the structure of the website. This improves the usability of the website because there is a reduction in confusion between where to find the character blogs and the theory blogs. A couple of other important useful links are also added.

The main blog page serves as a means of collecting all the blogs on a particular theme in one place. There are two. One for theory blogs and another for character blogs. Because there will be so many blogs on a page, I have decided to display only short excerpts of each blog (a few lines) followed by a "read more." link to the full blog. This helps readers find what they want easier and therefore improves the usability of the site.
